# 📊 Timeline Analytics Dashboard

## 🧩 Project Name
Keen-Keeper

---

## 📌 Short Description
A modern React-based dashboard that visualizes and analyzes user interaction timelines (calls, texts, and videos) using dynamic filtering and interactive charts. It provides real-time insights into communication patterns with a clean and responsive UI.

---

## 🛠️ Technologies Used
- React.js
- Next.js
- JavaScript (ES6+)
- Tailwind CSS
- Recharts (Data Visualization)
- React Context API

---

## ✨ Key Features

### 1. 📈 Interactive Data Visualization
Displays user interaction data using dynamic Pie Charts built with Recharts for clear insights.

### 2. 🔍 Smart Filtering System
Allows users to filter timeline data by interaction type (Call, Text, Video) in real time.

### 3. ⚡ Context-Based State Management
Uses React Context API to efficiently manage and share timeline data across components without prop drilling.
